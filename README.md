 Windows USB Forensic Investigation

A digital forensics project involving the analysis of a Windows USB / Windows filesystem image using Autopsy, Chainsaw, and Windows artifacts to uncover email evidence, user activity, and suspicious event logs.

This investigation focuses on analysing a Windows system to identify:

- Email artefacts and Gmail activity  
- Browser history related to Gmail logins  
- Security event logs tied to Microsoft accounts  
- Suspicious system activity and log entries  
- Evidence of user behaviour on the machine  

**Tools used:**

- Autopsy 4.22.1  
- Chainsaw  
- Windows Event Logs  
- CSV evidence (service_installation.csv)
